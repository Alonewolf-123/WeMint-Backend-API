#WeMint API

- install mongodb
  https://www.mongodb.com/docs/manual/installation/
  create wemint_db database in mongodb
- install node-modules
  npm install
 - run dev mode
  npm run dev
 - run prod mode
  npm run start
 - api documentation
  http://localhost:8080/api-docs
